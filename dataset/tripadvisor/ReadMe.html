Data Name: TripAdvisor v2

Data Descriptions:
This data was scripted from online reviews on tripadvisor.com. There is only one context: trip type (Family, Couples, Business, Solo travel, Friends). Other features about users and hotels are available. The data set is pretty sparse in ratings and contexts: 14175 ratings, 2731 users, 2269 hotels.
